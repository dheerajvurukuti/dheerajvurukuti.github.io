AgWeatherNet is Washington State University's flagship agricultural weather platform, providing real-time data from a network of hundreds of weather stations across Washington State to growers, agronomists, and researchers. I led the frontend modernization of this high-traffic platform — serving 29K+ active users — migrating a legacy codebase to React + TypeScript (Vite) to improve maintainability, bundle efficiency, and long-term scalability.

The platform delivers current and historical weather observations, interactive station maps, crop-specific forecasts, and a suite of agronomic decision tools covering irrigation scheduling, frost risk, degree-day accumulation, and spray condition windows. The modernized frontend significantly improved page load performance and developer experience, enabling faster iteration on new features.
