I led the frontend modernization of AgWeatherNet, WSU's flagship agricultural weather platform serving 29K+ active users, migrating a legacy codebase to React + TypeScript (Vite) to improve maintainability, bundle efficiency, and long-term scalability.

### The Platform

AgWeatherNet provides real-time weather data from hundreds of monitoring stations across Washington State, serving growers, agronomists, extension specialists, and researchers. The platform's feature set covers current and historical weather observations, interactive station maps, crop-specific forecasts, and agronomic decision tools for irrigation scheduling, frost risk assessment, degree-day accumulation, and spray condition windows.

### The Modernization

The legacy codebase presented significant maintainability and performance challenges that were slowing feature development and accumulating technical debt. Migrating to React + TypeScript introduced type safety, component reusability, and a modern build system through Vite that substantially reduced bundle sizes and improved page load performance.

The new component architecture makes it easier to iterate on features without breaking existing functionality, which matters for a platform that growers depend on daily during critical growing season decisions.
