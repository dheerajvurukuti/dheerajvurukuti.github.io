I developed this framework as part of my master's thesis at Washington State University, addressing one of the fundamental challenges in real-world malware detection: concept drift. As mobile threat landscapes evolve, classifiers trained on historical data inevitably degrade. The core question my research tackles is not just "is this malware" but "how confident should the system be in that answer", and what to do when that confidence is low.

The Approach: Deep Kernel Learning with Gaussian Processes

My framework integrates Gaussian Processes into a Deep Kernel Learning setup, giving the detection model the ability to quantify its own uncertainty alongside each prediction. When uncertainty is high, the system abstains rather than guessing, a selective classification strategy that significantly reduces misclassifications in adversarial and evolving environments.

The neural network component learns a rich feature representation from raw malware data, while the Gaussian Process layer produces calibrated probability estimates with explicit uncertainty bounds on top of those features. This combination scales better than standalone GPs while retaining the probabilistic guarantees that make uncertainty quantification meaningful.

Evaluation

Evaluated on two mobile malware datasets, KRONODROID and TRANSCENDENT, both of which include temporal drift where the test distribution shifts meaningfully from training data over time:

  • 31% improvement in AUT-F1 over state-of-the-art baselines
  • Ablation studies confirmed uncertainty quantification as the primary driver of performance gains under distribution shift
  • Class imbalance addressed through targeted sampling strategies, improving robustness on rare malware families

Why It Matters

In security applications, the cost of a wrong prediction is asymmetric. A missed malware classification can have serious consequences while a false positive creates friction. A system that knows when it doesn't know, and responds accordingly, is far more trustworthy in deployment than one that always forces an answer. This work shows that principled uncertainty quantification can be made practical at scale.
