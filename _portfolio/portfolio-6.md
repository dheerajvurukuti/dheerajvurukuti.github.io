I developed AWN Smart Farms as part of the AgWeatherNet engineering team, building a precision agriculture platform that extends the AgWeatherNet ecosystem into distributed IoT sensor management. Where the main AgWeatherNet platform focuses on regional weather station data, Smart Farms is built around field-level IoT sensors: soil moisture, temperature, and microclimate devices deployed directly in growing areas.

### What I Built

The frontend is built with React + TypeScript (Vite), centered on real-time data visualization across sensor networks:

- Geospatial visualization of sensor deployments across field layouts
- Live-updating dashboards pulling real-time data feeds from distributed IoT devices
- Automation workflows for alert configuration and threshold monitoring
- Scalable analytics for processing and displaying historical sensor data across a season

### Who Uses It

Smart Farms targets growers running precision agriculture programs who need field-level granularity beyond what a regional weather station provides. The dashboard surfaces actionable insights rather than raw sensor readings, helping growers make timely decisions around irrigation, crop health, and resource allocation across their operations.
